# Sega Superstars Tennis Recompiled

> NOTES
> This uses ReXGlue v0.3.
> It is playable from start to finish but you may get slowdowns.

Recompiled using the ReXGlue SDK:
https://github.com/rexglue/rexglue-sdk

# Setup

1. Download 'segasuperstarstennisrecomprelease.exe' from releases tab
2. Extract the Sega Superstars Tennis files into an `assets` (must be in same directory as .exe)
3. Run .exe and enjoy!

# Build
Follow the instructions here [ReXGlue Wiki](https://github.com/rexglue/rexglue-sdk/wiki)
